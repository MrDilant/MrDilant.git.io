# Portafolio Web - Dilant Murillo

## Descripción
Este proyecto es un portafolio web personal desarrollado con HTML, CSS y JavaScript.
Incluye información sobre mí, mis proyectos, un formulario de contacto y una sección adicional dedicada a mis mascotas.

El objetivo es presentar de forma clara mis habilidades en desarrollo frontend y la estructura de una página web funcional.

## Estructura del proyecto
- index.html → Página principal del portafolio
- pets.html → Página de mascotas
- css/ → Archivos de estilos organizados
- javascript/script.js → Funciones interactivas
- img/ → Imágenes

## Secciones
- Sobre mí
- Proyectos
- Contacto
- Mascotas

## Funcionalidades
- Menú de navegación responsive
- Botón hamburguesa
- Interacciones con JavaScript
- Formulario de contacto
- Botón flotante de WhatsApp
- Uso de íconos

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
- Bootstrap Icons

## Uso
1. Descargar o clonar el repositorio
2. Abrir el archivo index.html en el navegador

## Objetivo
Practicar el desarrollo de interfaces web, organización de estilos y uso básico de JavaScript.
